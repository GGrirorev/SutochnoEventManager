# Модуль "Комментарии"

## Описание
Добавляет возможность оставлять комментарии к событиям аналитики для обсуждения деталей реализации, вопросов и предложений.

## Функциональность
- Просмотр комментариев к событию
- Добавление новых комментариев
- Отображение автора и даты комментария

## Использование
Компонент Comments отображается на вкладке "Описание" в модальном окне события.

```tsx
import Comments from "@/plugins/comments";

<Comments eventId={eventId} />
```

## API
- GET /api/events/:id/comments - Получить комментарии события
- POST /api/events/:id/comments - Добавить комментарий

## Конфигурация
Модуль не требует дополнительной конфигурации.

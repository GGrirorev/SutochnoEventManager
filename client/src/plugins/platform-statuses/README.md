# Статусы платформ

Модуль для управления статусами внедрения и валидации событий по платформам.

## Описание

Модуль позволяет отслеживать состояние внедрения и валидации аналитических событий для каждой платформы (WEB, iOS, Android, Backend) с сохранением полной истории изменений.

## Возможности

- Статус внедрения: черновик, в разработке, внедрено, архив
- Статус валидации: ожидает проверки, корректно, ошибка, предупреждение
- История изменений статусов с датами
- Ссылки на задачи в Jira для каждой платформы
- Версионирование: каждая версия события имеет независимые статусы

## API

- `GET /api/events/:id/platform-statuses?version=N` - получить статусы для версии
- `PATCH /api/events/:id/platform-statuses/:platform` - обновить статус платформы

## Расположение

Компонент отображается на вкладке "Здоровье" в модальном окне события.

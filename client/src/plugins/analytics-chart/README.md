# График аналитики

Модуль для отображения графика событий аналитики в карточке события.

## Описание

Модуль получает данные о событии из системы аналитики Matomo и отображает график количества событий за последние 30 дней с разбивкой по платформам.

## Возможности

- Загрузка данных из Matomo API по каждой платформе
- Линейный график с динамикой событий за 30 дней
- Отображение данных по платформам: WEB, iOS, Android
- Цветовая кодировка платформ
- Интерактивный tooltip с детальной информацией

## Требования

Для работы модуля необходимо настроить интеграцию с Matomo:
- Переменная окружения `ANALYTICS_API_TOKEN` с токеном доступа к Matomo API

## Расположение

График отображается на вкладке "Здоровье" в модальном окне события.

## Конфигурация

Модуль не требует дополнительной конфигурации.

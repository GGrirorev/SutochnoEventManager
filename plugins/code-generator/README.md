# Генератор кода Matomo

Плагин для генерации примеров кода отправки событий в Matomo.

## Описание

Этот плагин добавляет в детали события раздел с готовыми примерами кода для отправки аналитических событий в Matomo. Код генерируется автоматически на основе категории, действия и имени события.

## Поддерживаемые платформы

- **Web** - JavaScript код для веб-приложений (`_paq.push`)
- **iOS** - Swift код для iOS приложений (`MatomoTracker.shared.track`)
- **Android** - Kotlin/Java код для Android приложений (`TrackHelper.track`)

## Использование

После включения плагина в разделе "Модули" администратора, в модальном окне деталей события появится новый раздел "Код для Matomo" с готовыми примерами кода для каждой платформы события.

## Конфигурация

В файле `manifest.json` можно настроить:

- `showForPlatforms` - список платформ для которых генерировать код

## API

Плагин экспортирует React-компонент `MatomoCodeGenerator`, который принимает:

```typescript
interface Props {
  event: {
    category: string;
    action: string;
    name?: string;
    platforms: string[];
    properties?: Array<{ name: string }>;
  }
}
```
